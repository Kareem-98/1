# موقع دعوة زفاف كريم وسارة

موقع دعوة زفاف أنيق ومتكامل (ملف HTML واحد) بلغة عربية وتصميم فاخر.

## معاينة مباشرة
https://Kareem-98.github.io/1

## مميزات الموقع
- تصميم responsive (موبايل أولاً)  
- مؤثرات حركية عالية الجودة عبر GSAP  
- شاشة دخول تفاعلية (ظرف قابل للفتح)  
- عدّـة تنازل حيّة حتى يوم 22 سبتمبر 2026  
- خريطة Google Maps مدمجة لموقع دار جاردينيا – الحرب الكيميائية  
- خلفية موسيقية رومانسية (تبدأ بعد نقر الزائر)  
- لا حاجة لأي إطار عمل أو مكتبات ثقيلة  

## طريقة الاستخدام المحلية
1. افتح ملف `index.html` في أي متصفح حديث  
2. انقُر على الظرف … تغادر الشاشة الافتتاحية وتنطلق الموسيقى  

## التعديلات السريعة
- لتغيير التاريخ: عدّل السطر `const wedDate = ...` داخل الوسم `<script>`  
- لتغيير الموقع أو خلفية الموسيقى: غيّر الروابط مباشرة فى نفس الملف  

## الترخيص
مشروع مفتوح المصدر بشكل عام – يمكن استخدامه ونسخه أو تعديله بحرية.
